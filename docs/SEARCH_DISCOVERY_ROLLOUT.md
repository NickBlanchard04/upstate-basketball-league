## Google Search Console

Use only the `nickblanchardbusiness@gmail.com` Google account.

1. Open the `upstatebasketballleague.com` domain property.
2. Submit `https://upstatebasketballleague.com/sitemap.xml` under Sitemaps.
3. Use URL Inspection and request indexing for the homepage, schedule, teams, news, about, and sponsors pages.
4. Inspect one boys team page and one girls team page after the first deployment.
5. Check Page indexing and Core Web Vitals weekly during launch month, then monthly.

Submitting a sitemap or requesting indexing does not guarantee placement or ranking. It helps search engines discover and re-check the pages.

## Bing and AI search discovery

The GitHub Pages release workflow sends every sitemap URL to IndexNow after deployment. Confirm that the `notify-indexnow` job succeeds in the release run. The sitemap and `llms.txt` remain publicly available for search and answer-engine crawlers.
