# UBL Team Representative Onboarding Emails

Send these from `info.upstatebasketballleague@gmail.com`. Replace bracketed fields before sending and never include passwords or owner-account credentials.

## 1. Request the representative's Google account

**Subject:** UBL score system access for [Program]

Hi [Name],

The UBL uses a simple score portal so each program can submit final scores without editing the league website. Once a score is accepted, the website automatically updates the result, standings, homepage game information, and playoff seeding.

Please reply with the exact Google email address that should receive access. We recommend one primary score submitter per program. This account will receive access only to the coach score portal, not the private commissioner control panel.

Before league-wide access is activated, we will send a short practice game that should take about five minutes.

Thanks,
Andy Walts
UBL Commissioner
