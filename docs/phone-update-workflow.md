# UBL Phone Update Workflow

League leadership control sheet: [UBL League Control Panel - 2026-27](https://docs.google.com/spreadsheets/d/1AVo5oRxSCFuTrkCK4MA30vT5u_6XuLAXIvyaqnWgcRE/edit)

## Coach: submit a final

1. Open the private link for your team's score portal.
2. Find the game card.
3. Enter the away and home scores in the yellow cells.
4. Check `Submit` and read `Status`.

The portal records your name automatically. If Status starts with `Review`, verify the numbers and check Submit again. Use `Correction Request` for a published score that needs commissioner approval.

## Commissioner: update a game

1. Open `Commissioner Dashboard` in the control panel.
2. Edit the game row using the team, venue, and status dropdowns.
3. For a final or forfeit, enter both scores before selecting the final status.
4. Confirm the `Game updated everywhere` message.

Use `Corrections Queue` to approve or reject coach corrections. Do not unhide backend sheets for normal work.

## Safety checks

- Final scores must be nonnegative whole numbers and cannot be tied.
- Public Notes are visible on the website.
- If the website shows `Using backup schedule`, do not re-enter data. The live feed is temporarily unavailable.
- `Operations Alerts` records hourly overdue-score and feed checks; daily backups run automatically.
