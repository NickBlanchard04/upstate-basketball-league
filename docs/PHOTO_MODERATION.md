# UBL Photo Submission and Moderation

Public gallery photos must never publish automatically. The league inbox is the intake queue, and the website gallery contains only files approved by a league administrator.

## Submission requirements

Each submission should include:

- Team and division
- Game or event date
- Submitter's full name
- Confirmation that the submitter has permission to share the photos

## Review process

1. Open attachments only from an identifiable team representative, coach, parent, photographer, or league contact, and only when the files are JPG, PNG, or WebP images.
2. Confirm the images are UBL-related and that the submitter provided the required permission statement.
3. Reject images containing inappropriate, unrelated, private, or unsafe material. Do not forward rejected files.
4. Check for duplicate or near-duplicate photos before approving a submission.
5. Move approved originals into a private Drive folder organized by team and season.
6. Publish only optimized copies through the website repository. Never expose the intake folder or grant public edit access.
7. Keep the submitter's email and approval decision as the audit record.

## Removal requests

Send removal requests to Info.upstatebasketballleague@gmail.com. Remove the public image promptly while the request is reviewed.

## Future form

If UBL replaces email intake with a Google Form, require Google sign-in, restrict uploads to image files, limit file size, collect the same permission statement, and keep the destination folder private. Form uploads must still pass the human review process above.
